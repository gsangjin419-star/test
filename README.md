# Stable Arcade Tetris 🎮

> 웹 기반 안정화 테트리스 게임  
> 최신 브라우저에서 실행 가능, 모바일/데스크탑 모두 지원

---

## 📝 소개

Stable Arcade Tetris는 클래식 테트리스를 현대 웹 환경에 맞게 구현한 버전입니다.  

주요 특징:

- 안정화된 충돌 및 회전 로직 (벽 끼임 방지)
- 고스트 블록 표시
- 하드 드롭 및 소프트 드롭 지원
- 점수, 최고 점수 저장 (localStorage)
- T-Spin / 콤보 점수 계산
- 레벨 시스템
- 모바일 터치 버튼 지원
- 스킨 변경 기능
- 간단한 배경음 및 효과음

---

## ⚡ 게임 방법

### 키보드

| 키 | 기능 |
|----|------|
| ← / → | 좌/우 이동 |
| ↓ | 소프트 드롭 |
| ↑ | 회전 |
| Space | 하드 드롭 |
| P | 일시정지 |

### 모바일
- 화면 하단의 버튼으로 좌/우 이동, 회전, 소프트/하드 드롭 가능

---

## 🎨 스킨

- 기본, 네온, 모노 3가지 스킨 제공
- 게임 중 스킨 버튼 클릭으로 변경 가능

---

## 🏆 점수 시스템

- 한 줄 제거: 100~800점
- T-Spin 보너스: +400점
- 연속 줄 제거 콤보: +50점/콤보
- 최고 점수는 브라우저에 저장

---

## 🛠 기술 스택

- HTML5 Canvas
- CSS3
- JavaScript (ES6)
- Web Audio API (사운드 및 BGM)
- localStorage (최고점수 저장)
